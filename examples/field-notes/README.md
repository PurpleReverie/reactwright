# examples/field-notes

A Tufte-style essay with sidenotes in the outside margin. Exercises
`<sidenote>` + `<sidenote-area>`, `<defs>`/`<def>` glossary, `<ref>`,
`<toc>`, `<index>`, drop caps on opener paragraphs, and declarative
`<font>` loading.

## Run it

```sh
pnpm --filter @example/field-notes mockup
```

Outputs `build/mockups/field-notes.{html,pdf}`.
