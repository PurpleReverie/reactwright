# examples/report-sample

A technical/business report mockup against
`@reactwright/template-report`. Includes an executive summary, four
numbered sections, a figure, a table, inline citations, and a numeric
bibliography. All facts, figures, and findings are invented for layout
purposes.

## Run it

```sh
pnpm --filter @example/report-sample mockup
```

Outputs `build/mockups/report-sample.{html,pdf}`.
