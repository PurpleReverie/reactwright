# examples/essay-sample

A short MLA-style academic essay. Demonstrates the
`@reactwright/template-essay` template against a single-file content
document: three sections, one block quote, inline citations, and a
Works Cited list.

## Run it

```sh
pnpm --filter @example/essay-sample mockup
```

Outputs `build/mockups/essay-sample.{html,pdf}`.
