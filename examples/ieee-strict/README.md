# examples/ieee-strict

A strict IEEE conference paper authored against
`@reactwright/template-ieee`. Single-file content; the template lives
in the package. Demonstrates `createBibliography(...)` for typed
citation keys and the `IEEEFrontMatter` helper.

## Run it

```sh
pnpm --filter @example/ieee-strict mockup
```

Outputs `build/mockups/ieee-strict.{html,pdf}`.
