# examples/book-sample

A short-novella-shaped book sample rendered with
`@reactwright/template-book`. Demonstrates the trade-paperback
template against a single-file content document: title page,
copyright + dedication front matter, three chapters with chapter
headings and a sub-section, one block quote, and an afterword +
acknowledgments back matter. All names, places, and prose are invented
for layout purposes.

## Run it

```sh
pnpm --filter @example/book-sample mockup
```

Outputs `build/mockups/book-sample.{html,pdf}`.
