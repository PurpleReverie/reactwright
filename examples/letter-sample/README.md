# examples/letter-sample

A one-page formal business letter rendered with
`@reactwright/template-letter`. Demonstrates every region of the
template: letterhead, date, addressee, subject, salutation, two body
paragraphs, closing, and signature block. Names, addresses, and
subject line are invented for layout purposes.

## Run it

```sh
pnpm --filter @example/letter-sample mockup
```

Outputs `build/mockups/letter-sample.{html,pdf}`.
