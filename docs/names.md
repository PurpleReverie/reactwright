# The Name Book

A short ledger of every name considered before this project became
**Reactwright**, kept here so that the road not taken is at least written
down somewhere.

The hunt ran across four naming traditions, each of which produced
something worth recording even when it didn't survive the vote.

---

## I. The typography heritage (single words)

The first wave reached for words that have lived in printing shops
for centuries. Almost every one of them was already claimed on npm,
which is itself a small piece of evidence about how much of the
ecosystem already knows it cares about typesetting.

| Word | Meaning | npm |
|---|---|---|
| folio | a leaf of paper; a book page numbered front and back | taken |
| galley | a typesetter's proof column | taken |
| forme | the locked-up frame of metal type ready for the press | taken |
| quire | a set of folded sheets (the unit a printer thinks in) | taken |
| recto | the front side of a leaf (verso is the back) | taken |
| codex | the ancient form that replaced the scroll | taken |
| imprint | the publisher's mark + the act of pressing type to paper | taken |
| vellum | calfskin parchment | taken |
| pica | a typographic measure, twelve points | taken |
| sheaf | a bundle of bound sheets | taken |
| quoin | the wedge that locks a forme of type | **free** |
| chapbook | a small cheaply-printed booklet | **free** |
| foliate | to put leaves on (as a verb) | **free** |
| libella | Latin for "little book" | **free** |

The verdict was that the single-word names had already been spoken
for by people who, decades ago, also liked the smell of a printshop.
We tipped our hats and moved on.

## II. The scholarly bin (Latin, Greek, made-up brand)

The second wave invented words rooted in classical languages or in
the "modern indie brand" tradition — Astro, Solid, Photon, Vite —
where the name itself is invented but feels rooted.

| Word | Provenance | npm |
|---|---|---|
| folion | folio + -on; an invented brand on a known stem | **free** |
| libella | Latin: little book | **free** |
| calamus | Latin: reed pen | **free** |
| tractus | Latin: a written treatise | **free** |
| opuspress | opus + press | **free** |
| typewright | type + wright (Old English: a craftsman) | **free** |
| pagecraft | the craft of the page | **free** |
| pageworks | page + works (in the Wolfram / MathWorks tradition) | **free** |
| pageleaf | a poetic compound: page and leaf are different things | **free** |
| inkfold | ink + fold; bound pages and printer's ink | **free** |
| inkly | ink + -ly; a friendly suffix on a fundamental | **free** |

The shortlist out of this round was **Folion**, **Typewright**,
**Pagecraft**, and **Libella** — four names that each carry the work
already done by someone in a previous century.

## III. The React combinations

The shortlist needed React in it. The point of this project is that
the document is authored in React; the name should declare so. The
combinations were obvious in retrospect:

| Compound | Meaning | npm |
|---|---|---|
| reactype | react + type | taken |
| reactpress | react + press | taken |
| reactpage | react + page | taken |
| react-bound | react + bound | taken |
| reactfolio | react + folio | **free** |
| reactfolion | react + folion | **free** |
| reactleaf | react + leaf | **free** |
| reactbound | react + bound (unhyphenated) | **free** |
| reactprint | react + print | **free** |
| reactcraft | react + craft | **free** |
| reactwright | react + wright (craftsman) | **free** |
| reactlibella | react + libella | **free** |

## IV. The vote

The final four were:

1. **Reactwright** — react + wright. Wright is the Old English
   craftsman suffix: a playwright writes plays, a wheelwright makes
   wheels, a typewright (in this branch of the history) typesets
   pages. Reactwright is a craftsman of pages, in React.
2. **Reactfolion** — modern invented brand on the folio stem; the
   `-on` ending puts it in the company of Astro, Solid, Photon.
3. **Reactlibella** — the most scholarly. Latin for "little book"
   with React attached. Best for an academic positioning.
4. **Reactcraft** — broadest, simplest, most generic of the four.

The vote went to **Reactwright** because it is the only name that
compactly carries all three of the things this project wants to be:
that it is React (the substrate); that it is a craft (typography is
a craft, not an algorithm); and that it has heritage (every artisan
suffix `-wright` ever attached to a profession survived because the
work was patient enough to deserve a name).

---

If you are reading this, you have found the easter egg. The name was
not chosen lightly. Most software is named in a hurry; this one
wasn't.
